# PeglinAssetReader
This application was designed to generate an html page from the game files to make it easier to identify relics when watching streams. Feeds https://archindivide.github.io/PeglinAssetReader/ with the current version's info.

## Future planned features

- grouping by (or indicator for) rarity/drop pool
- search bar to quick filter items (helpful for bad mobile experience)
- different UI for mobile experience/limited screen space
- other in game info (curses, balls, status effects, etc)
